Kurzlebige Schlüssel / Forward Secrecy

{{if .EphemeralKeysSupported}}

Gut Dein Client bietet Cipher-Suiten, die kurzlebige Schlüssel (Ephemeral Keys) nutzen. Das bedeutet, er kann Forward Secrecy gewährleisten, wenn der Server das ebenfalls unterstützt. Das erhöht die Abhörsicherheit deutlich, insbesondere gegenüber passiven Lauschern, die große Mengen verschlüsselter Daten speichern, bis sich ihre Angriffsmöglichkeiten verbessern.

{{else}}

Ausbaufähig Keine der von deinem Client unterstützten Cipher-Suiten bietet kurzlebige Schlüssel (Ephemeral Keys). Das bedeutet, dein Client kann keine Forward Secrecy gewährleisten. Ohne die können passive Angreifer verschlüsselt übertragene Daten speichern, um sie später zu dechiffrieren, wenn ihre Angriffe schnell genug arbeiten oder sie anderweitig etwa an den geheimen Server-Schlüssel kommen. Das geschieht tatsächlich.

{{end}}
